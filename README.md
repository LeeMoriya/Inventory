# Inventory

![Preview](https://media.giphy.com/media/xeqvH6E6tBfswpJ5Wk/giphy.gif)

A Rain World mod that adds a configurable Inventory for storing additional items.

## Controls
Map - Open inventory

Movement - Move cursor

Jump - Store/retrieve

Pickup - Toggle Map
